月比对数据判定程序
=================
> author:liziru &zzt
> date:2021-05-04

### 程序的目的
方便李姿儒的工作，更快判断数据是否在容差范围内
### 程序流程
读取允差配置文件(excel)，读取需要判定的源素材数据，逐个数据判断允差是否在范围内，并输出一个判定结果文件(excel)
### 程序启动方式
> 配置：若不配置使用程序内置的默认地址
```
python main.py --input-data/-i [输入的素材数据地址] --setting-data/-s [允差配置文件地址] --output-data/-o [输出文件地址]
```
